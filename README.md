# 🌤 Weather App (Python CLI Project)

## 📌 Description

The **Weather App** is a Python command-line application that provides real-time weather information for any city using the OpenWeatherMap API.
Users can search for weather details such as temperature, weather condition, humidity, and wind speed. The program also maintains a search history and displays the time when the weather information was checked.

This project demonstrates Python fundamentals, API integration, error handling, and command-line interface design.

---

## 🚀 Features

* 🔍 Search weather by city name
* 🌡 Temperature display in **Celsius (°C)** and **Fahrenheit (°F)**
* ☁ Weather condition display
* 💧 Humidity information
* 🌬 Wind speed display
* 🕒 Date and time of weather check
* 📜 Search history tracking
* 📋 Menu-based command-line interface
* ⚠ Error handling for invalid city names or network issues
* 🎨 Colored CLI output using Colorama
